# C#/.NET API & UI tests portfolio
Welcome in my framework created from scratch. This is a showcase for automated tests performed on public demo pages: 
- API: https://restful-booker.herokuapp.com
- UI: https://sauce-demo.myshopify.com

It is still under development - more cases are coming.

---
## Tech stack

**API automation**
- NUnit
- HttpClient
- API Client Pattern
- Test Data Builder

**UI automation**
- NUnit
- Playwright for .NET
- Page Object Model / Pattern

**CI/CD**
- YAML config for GitHub actions
